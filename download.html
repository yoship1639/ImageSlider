ダウンロード

ここからダウンロードできます。
解凍して好きなフォルダに配置してください。
使い方は解凍したフォルダの中にあるreadme.txtを読んでください。

Windows 32bit版 ImageSlider_v2.1_win32.zip

Windows 64bit版 ImageSlider_v2.1_win64.zip


ソフトウェアを実行するには、.Net Framework 4.5がインストールされている必要があります。
実行できない場合はこちらからダウンロードし、インストールしてください。

履歴

2016/02/09 - ver2.1
  • 更新情報の自動チェックを改良
  • PixivAPIが再び利用可能
  • PicasaAPIを新たに追加
  • ImgurAPIを新たに追加
2016/01/22 - ver2.0
  • 画像検索APIの扱い方を一新
  • 画像検索APIを自動で更新するように仕様変更
  • 画像検索APIを自動で更新するか決める設定項目を追加
2016/01/21 - ver1.4
  • GoogleImageSearchのサービスが終了したため利用不可
  • PixivAPIのサービスが終了したため利用不可
  • 新たにTiqavAPIを追加
  • 新たにPixaboyAPIを追加
2015/05/24 - ver1.3
  • ウィンドウを常に最前面化する機能を追加
  • ウィンドウを常に最前面にするか決める設定項目を追加
  • PixivAPIの仕様変更のため一時的に利用不可
2015/01/19 - ver1.2
  • 検索履歴の一覧リストを検索ワードボックスの隣に追加
  • 検索履歴をすべて削除するボタンを追加
  • 検索ワードにオートコンプリート機能を追加
  • 検索ワード入力時にオートコンプリートを行うか決める設定項目を追加
2015/01/18 - ver1.1
  • ボタン等のヒントを表示するかを選択できる項目を追加
  • 検索APIの表示名を少し修正
2014/12/07 - ver1.0
  • 公開
